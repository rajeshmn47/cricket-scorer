## Ballbyball | Your cricket partner [Live URL](https://ballbyball.vercel.app)

A website where you can count your score along with handling players and teams

### Tech Stack Used
- React.js
- Next.js
- APIs with Next.js
- Prisma
- Tailwind CSS
- shadncn-ui
- React Query
- Zod
- Clerk auth
- Recharts
- React hook form
